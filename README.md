## 📋 Sobre o Projeto!


Landing page interativa temática do **Homem-Aranha**, construída do zero com **HTML, CSS e JavaScript puro** — sem frameworks, sem bibliotecas externas.

O projeto explora animações avançadas com Canvas API, interatividade com o mouse e efeitos visuais com CSS puro, tudo no estilo sombrio e vermelho do traje clássico do herói.

## ✨ Funcionalidades

- 🖱️ **Cursor personalizado** — cursor animado que reage ao clique
- 🕸️ **Teia no fundo** — rede de nós conectados que seguem o movimento do mouse em tempo real
- 🦸 **Personagem SVG** — Homem-Aranha desenhado em SVG puro com animação de flutuação
- ⚡ **Lança-Teia interativo** — clique em qualquer lugar da seção para disparar teias com efeito radial
- 📊 **Barras de progresso animadas** — ativadas ao rolar a página via Intersection Observer
- 📱 **Responsivo** — adaptado para diferentes tamanhos de tela

## 🛠️ Tecnologias & Conceitos

| Tecnologia | Uso no Projeto |
|---|---|
| **Canvas API** | Animação das teias e nós flutuantes em tempo real |
| **requestAnimationFrame** | Loop de animação performático |
| **Intersection Observer API** | Barras de progresso ativadas no scroll |
| **CSS Keyframes** | Animações de float, fadeUp e bounce |
| **CSS clip-path** | Formas geométricas nos botões e cards |
| **CSS Gradients** | Texto com gradiente via background-clip |
| **SVG inline** | Personagem desenhado com código puro |
| **Math.cos / Math.sin** | Trigonometria para o efeito radial da teia |
